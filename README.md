Στόχος του έργου αυτού είναι η δημιουργία ενός οχήματος, Line-follower, το οποίο θα μπορεί αυτοματοποιημένα να ακολουθεί μια "πίστα" που αποτελείται από μία μαύρη λωρίδα. Εδώ θα βρείτε τα απαραίτητα υλικά καθώς και τις οδηγίες για την δημιουργία του οχήματος.

  Υλικα:
  1x Raspberry Pi Pico,
  1x Driver,
  2x Motors,
  2x IR Sensors,
  1x Breadboard,
  1x AA 4-cell Battery Pack,
  4x AA Batteries,
  1x Power bank,
  1x USB to micro USB cable,
  Jumper Wires,
  1x 1kohm Resistor,
  1x Button,
  ~50 Pin Header Male, 
  2x Tires,
  1x 360° Rotation Wheel,
  1x 200cm² Sturdy Cardboard,
  Hot Glue,
  Soldering Iron.

Για το σχήμα και την εμφάνιση του οχήματος σας αφήνουμε να σκεφτείτε και να δημιουργήσετε ότι σας αρέσει εσάς. Πριν ξεκινήσετε όμως, δημιουργήστε ένα σχέδιο για το πώς θα δημιουργήσετε και που θα τοποθετήσεται το κάθε εξάρτημα, σιγουρεύοντας πως όλα θα χωρέσουν χωρίς προβλήματα. Το κυρίως βάρος(μπαταρίες) του οχήματος θα πρέπει να βρίσκεται στο πίσω μέρος του οχήματος πάνω από τις ρόδες. Οι αισθητήρες θα πρέπει να είναι τοποθετημένοι στο μπροστινό μέρος του οχήματος. Όσο πιο κοντά βρίσκονται οι ρόδες στους sensors τόσο πιο "γρήγορη" θα είναι η αντίδραση και επαναδιώρθωση της τροχιάς.

1) Κολλήστε το Raspberry Pi Pico και τα Male Header Pins χρησιμοποιώντας το Soldering Iron. Αυτή η διαδικασία χρειάζεται μεγάλη προσοχή για να μην καεί το Raspberry Pi Pico ή να ενωθούν 2 εξοχές που δεν θα έπρεπε. Εάν δεν έχετε εμπειρία βρείτε κάποιον που γνωρίζει πως να το κάνει. Αφότου ολοκληρωθεί αυτό μπορεί να μπει το Raspberry Pi Pico πάνω στο breadboard για να γίνουν οι απαραίτητες συνδέσεις με τα υπόλοιπα εξαρτήματα.
2) Κάνετε τις κατάλληλες κολλήσεις με το soldering iron για να συνδέσετε την μπαταριοθήκη στο Driver των κινητήρων.
